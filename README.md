LocalPdf 📄
A private, client-side PDF toolkit. Everything runs in your browser — no uploads, no servers.
Tools

Merge — Combine multiple PDFs
Split — Break a PDF into separate files
Extract — Save specific pages
Reorder — Drag and drop pages
Rotate — Rotate pages by angle
Watermark — Add custom text watermarks
Page Numbers — Stamp page numbers
Compress — Reduce file size
Protect — Add a password
Unlock — Remove a password
PDF → Images — Export pages as PNG or JPEG

Usage
Just open LocalPDF.html in any modern browser. No install needed.
Privacy
All processing happens locally. Your files never leave your device.
